12. Consider the following model:

\(log(wage) = \beta_0 + \beta_1 educ + \beta_2 exper + \beta_3 tenure + u\)


  1. State the null hypothesis that another year of general workforce experience has the same effect on log wage as another year of tenure with the current employer.
  2. Test the null hypothesis against a two sided alternative at the 5% significance level, by constructing a 95% confidence interval. What do you conclude?


Solutions

  1. The null hypothesis of interest is \(H_0: \beta_2 = \beta_3\).

  2. Let \(\theta_2 = \beta_2 - \beta_3\). We replace \(\beta_2\) in the initial model, which gives:

Step 1: \(\theta_2 + \beta_3 = \beta_2\)

Step2: \(log(wage) = \beta_0 + \beta_1 educ + (\theta_2 + \beta_3) exper + \beta_3 tenure + u\)

Step3: \(log(wage) = \beta_0 + \beta_1 educ + \theta_2exper + \beta_3exper + \beta_3 tenure + u\)

After replacement, our model is as follows:

\(log(wage) = \beta_0 + \beta_1 educ + \theta_2 exper + \beta_3(exper + tenure) + u\)
